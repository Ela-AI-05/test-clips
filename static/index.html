Shot & scene clips

Shot + scene clips, with timed transcripts (Whisper). Start backend python -m app.main and open this page at http://127.0.0.1:8000 (not as a saved file). First Whisper run may download the speech model. Optional speaker tags: use a Hugging Face token with access to pyannote/speaker-diarization-3.1 and run the server with HF_TOKEN=… (names are generic Speaker 1 / 2, not cast names). Explain scene needs OPENAI_API_KEY in a project-root .env file or environment variable (default vision model gpt-4o-mini). Scene explanations include optional spoken narration via OpenAI TTS ( tts-1, voice OPENAI_TTS_VOICE).
